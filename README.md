# Curso de Introdução ao Python no Google Colab

Este repositório contém 10 aulas introdutórias de Python puro, organizadas em notebooks `.ipynb` para uso no Google Colab.

## Estrutura

1. Introdução ao Python e Google Colab
2. Operadores Matemáticos e Lógicos
3. Estruturas Condicionais
4. Loops e Repetições
5. Funções
6. Listas
7. Tuplas, Dicionários e Sets
8. Strings e Manipulação de Texto
9. Arquivos em Python
10. Tratamento de Erros e Modularização

## Como usar

Abra cada notebook no Google Colab e execute as células em sequência.

## Objetivo

Construir uma base sólida em Python antes de avançar para ciência de dados, automação, APIs, análise de dados e machine learning.
